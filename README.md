# Iconic Guitars
A CRUD web app that lets you search for iconic guitars based on your favorite genre and guitar mode using a cloud-deployed database of instruments. 
## Getting Started
[
Click Here](https://iconicGuitars.herokuapp.com/)
## Technologies Used
* HTML
* CSS
* JavaScript/jQuery
* Node.js
* MongoDB
* Mongoose
* Express
* Google Fonts

* Make of Guitar
